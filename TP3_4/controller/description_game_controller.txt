Ce script définit une application FastAPI qui permet de jouer à un jeu de bataille navale en ligne.

L'application expose plusieurs routes permettant de créer et de rejoindre des parties,
d'ajouter des navires à un champ de bataille, de tirer sur un navire adverse et de récupérer l'état d'une partie.
Les données sont transmises et reçues sous forme de requêtes HTTP et de modèles de données Pydantic.

L'application utilise également le framework Uvicorn pour être lancée sur un serveur.
Enfin, l'application gère les exceptions en retournant une réponse JSON avec un code d'erreur et un message d'erreur en cas d'exception levée.
